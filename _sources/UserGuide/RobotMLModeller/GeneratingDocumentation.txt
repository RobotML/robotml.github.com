Generating documentation
========================

After creating your model, you may want to generate documentation for it.
To do this :

1. Start the RobotML tool
2. Right click on the model root in the Project explorer then click on Generate Documentation 


.. figure:: images/RobotMLGendoc1.png
   :align: center
   :alt: generator menu
   
3. A popup will appear

.. figure:: images/RobotMLGendoc2.png
   :align: center
   :alt: generator view
   
4. Choose the language of your model (default is RobotML)
5. Choose the output path (default is your project path)
6. Choose the name of the generated file (default is name_of_your_model_doc)
7. Click on finish
