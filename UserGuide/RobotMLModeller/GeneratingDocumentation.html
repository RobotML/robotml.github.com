Generating documentationÂ¶

After creating your model, you may want to generate documentation for it. To do this :

  1. Start the RobotML tool
  2. Right click on the model root in the Project explorer then click on Generate Documentation
  1. A popup will appear
  1. Choose the language of your model (default is RobotML)
  2. Choose the output path (default is your project path)
  3. Choose the name of the generated file (default is name_of_your_model_doc)
  4. Click on finish
